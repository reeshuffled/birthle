# birthle

A Wordle-like game where you guess famous people's birthdays.

Idea from [Will & Rusty’s Playdate podcast](https://www.tiktok.com/t/ZTBvLfWDq/).

As far as I know, it didn't exist before them. There are some things that are similar in concept or name, but not in execution.

* [Birthdayle](https://unscrambleit.net/birthdayle/)
* [Celebrity Birthday Game](https://celebritybirthdaygame.com/)
* [Birthday Wordle](https://birthdaywordle.pages.dev/)
* [Pantehon Birthle](https://pantheon.world/game/birthle)
